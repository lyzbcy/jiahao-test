# 初心与使命 · 嘉豪值测试

> 本文件是整个项目的"宪法"。后续所有设计——题目怎么出、维度怎么分、本命嘉豪怎么判定、文案什么调性、美术用什么图——都必须回头对齐这份文档。有分歧时，以这里为准。

---

## 一、项目是什么（一句话）

**做一个 MBTI 式的趣味人格测试网站：用户答完若干道题，后台按维度计算出他/她的"豪意值"，然后像开盲盒一样，抽出一个专属的"本命嘉豪"——比如自在极意豪、上课全对豪——配上骚话评价 + 分享卡 + 云排行榜。**

部署在 GitHub Pages 静态网站，后端用在线数据库（Supabase）存排行榜和做题记录。

## 二、什么是"嘉豪"（必须先搞懂的文化底子）

做这个项目前，必须先理解"嘉豪"到底是什么梗，否则题目和标签会写成尬梗。

- **起源**：2024 年，一位身穿黑衣黑帽黑口罩的中学生，在教室里模仿 Alan Walker《The Spectre》"虚空打碟"的视频走红。视频被误标了名字"嘉豪"，于是"嘉豪"就成了所有这类行为的代名词。
- **本质**：调侃学生时代那些**爱装×、爱表现、自我感觉良好、中二病爆棚（尤其是中学男生）**的行为。"嘉豪"不是一个侮辱词，更像一个**集体自嘲勋章**——人人都可能是嘉豪，承认自己嘉豪反而可爱。
- **核心量化概念——豪意值**：衡量一个人有多"嘉豪"的数值。豪意值越高，越嘉豪。
- **豪型光谱（亚种）**：网友已经发展出一套"嘉豪学"分类，常见亚种：
  - **上课全对豪**：豪意值较低、相对普通的嘉豪。对答案时大喊一声"全对"的那种。
  - **自在极意豪**：最高境界，"身在豪中不知豪"，无意识就做出豪意值拉满行为的人（致敬《龙珠》自在极意功）。
  - 还有更多可衍生：黑客嘉豪、文学嘉豪、雨中嘉豪、赛级嘉豪、始祖嘉豪……
- **经典行为**：打空气拳 / 空气投篮 / 全黑连帽衫+口罩 / 模仿 DJ 打碟 / 雨中漫步 / 对答案喊"全对" / 刻意营造神秘感。
- **经典句式**："同龄人：xxx；我：xxx；就很……（捂脸笑）"

> 参考来源：百度百科"嘉豪"词条、游民星空《在"嘉豪梗"诞生两年后，全世界都成为了嘉豪》、知乎《质疑嘉豪，理解嘉豪，成为嘉豪》。

**一句话总结我们要传递的调性**：用最正经的"科学测试"外壳，包裹最沙雕的嘉豪梗——越一本正经越好笑。和 SBTI、MBTI 恶搞一脉相承。

## 三、为什么做这个（初心）

1. **追热点 + 玩梗变现**：2026 是"嘉豪元年"，梗完全爆发，正处在流量高位。赶在热度窗口做一个高传播性的测试工具，是有明确时效价值的内容资产。
2. **复用成熟的测试玩法骨架**：我们已经有一套"开盲盒式趣味测试"的成熟脚手架（参见 lyzbcy-测试题制作 skill）。换主题就能快速产出，嘉豪梗非常适合——它天生带"豪型分类 + 豪意值"的数值化结构，几乎是为这种测试量身定做的。
3. **做能火的内容**：盲盒标签 + 个人专属评价 + 社交裂变（"100 个人里只有 3 人抽到自在极意豪"）是经过验证的爆火公式。嘉豪梗本身的"人人都是嘉豪"特质，天然适合全民自测 + 朋友圈分享。
4. **沉淀可复用的"热点测试"生产线**：嘉豪值测试跑通后，这套模板（维度计分→标签判定→上云→部署）可以直接套到下一个热点梗上。

## 四、核心机制设计（豪意值怎么算、本命嘉豪怎么抽）

这是项目的工程核心，采用"**三层换算**"管线（和 MBTI、高考位次换算同构）：

```
用户答题（每题选一个选项）
  → ① 维度计分：把答案累加成几个维度分   ← 类比"考试得分"
  → ② 锚定换算：把维度分换算成"豪意值"   ← 类比"分数→位次"
  → ③ 标签判定：按豪意值 + 维度组合，挑出本命嘉豪（带稀有度） ← 盲盒引擎
  → ④ 输出：骚话评价 + 分享卡 + 上云排行榜
```

### 4.1 维度结构（初稿，写代码前可微调）

模仿 MBTI 用 4 个维度，但全部"嘉豪化"（用沙雕词替换正经词）：

| 维度代号 | 极 A（低分） | 极 B（高分） | 测的是什么 |
|---|---|---|---|
| **装**（Z） | 朴实无华（P） | 极致装杯（Z） | 表现欲、刻意营造感 |
| **二**（E） | 清醒克制（Q） | 中二爆表（E） | 中二病浓度、自我沉浸度 |
| **骚**（S） | 老实本分（L） | 花式整活（S） | 整活能力、行为花样 |
| **觉**（J） | 身在豪中（Z） | 浑然自觉（J） | 对自己嘉豪行为的自觉程度 |

四维各取一极，组成 4 字母"豪型代码"，例如 **PESJ**（朴实·中二·整活·自觉）。

> 关键洞察：**"觉"维度的设计是这个测试的灵魂**。它把"自在极意豪"（身在豪中不知豪，不自觉，J 维取"身在豪中"）和普通嘉豪（知道自己装×）区分开——这正是嘉豪梗最精髓的"无意识嘉豪"概念。

### 4.2 豪意值（总分）

四个维度高分项加权求和 → 一个 0~100 的"豪意值"。豪意值决定**稀有度大档**：

| 豪意值区间 | 稀有度 | 大致含义 |
|---|---|---|
| 0–30 | N（普通） | 嘉豪门外汉，基本正常人 |
| 31–55 | R（稀有） | 嘉豪苗子，偶尔发作 |
| 56–75 | SR（超稀有） | 资深嘉豪，日常豪意 |
| 76–95 | SSR（传说） | 赛级嘉豪，豪意值拉满 |
| 96–100 | UR（隐藏） | 自在极意豪，身在豪中不知豪 |

### 4.3 本命嘉豪（标签，开盲盒核心）

豪意值定稀有度大档，**四维代码**（P/Z·Q/E·L/S·Z/J）在档内挑出具体的"本命嘉豪"。豪华 14 款清单（12 常规 + 2 隐藏，参考嘉豪学亚种）：

**常规（公开，12 个）：**

| 本命嘉豪 | 稀有度 | 豪型画像 |
|---|---|---|
| 上课全对豪 | N | 对答案大喊"全对"的本分嘉豪，豪意入门 |
| 空气投篮豪 | N | 无实物运动的体育嘉豪，随时起跳 |
| 做题嘉豪 | R | 沉浸刷题的卷王嘉豪，用努力装杯 |
| 虚空打碟豪 | R | 致敬梗起源的 DJ 嘉豪，耳机一戴谁也不爱 |
| 雨中漫步豪 | R | 刻意营造氛围的文艺嘉豪，浪漫全靠自己演 |
| 黑衣口罩豪 | SR | 经典全黑穿搭的神秘嘉豪，原教旨嘉豪 |
| 文学嘉豪 | SR | 出口成章的装文嘉豪，金句比诗多 |
| 金融嘉豪 | SR | 动不动聊投资理财的精英嘉豪，余额没有术语多 |
| 黑客嘉豪 | SSR | 键盘敲得飞起的科技嘉豪，绿屏代码护体 |
| 兵王嘉豪 | SSR | 脑补自己是退役特种兵的硬核嘉豪，走路带风 |
| 球场嘉豪 | SSR | 空气拳+花式运球的运动嘉豪，全场最佳（幻想中）|
| 始祖嘉豪 | SSR | 一切嘉豪的源头，初代形态，传说中的男人 |

**隐藏款（2 个，彩蛋触发，不进公开概率榜）：**
- **自在极意豪**（UR）——豪意值爆表且"身在豪中不知豪"（"觉"维度取不自觉极 + 豪意值 96+），最高境界，致敬龙珠。
- **仙豪模式**（UR）——彩蛋触发（某题选了特定"觉醒"选项、或四维全取同一极等极端组合），传说之上的传说。

### 4.4 显示概率 ≠ 真实抽取概率（重要约定）

盲盒的真实分布由"用户答题 → 维度命中"决定，**不是随机抽卡**。给用户看的"显示概率"（如"3% 的人抽到自在极意豪"）是基于历史数据统计的营销文案，不是程序随机概率。这一点在文档和给用户的说明里都要写清楚，避免"概率造假"的质疑。

## 五、玩法流程（用户视角）

1. **首页**：嘉豪吉祥物迎接，一句话钩子——"你是哪种嘉豪？测测你的豪意值"，大按钮"开始测试"。
2. **做题**：12–16 道情境题，每题 2–4 个选项，单选。题目用嘉豪经典场景（教室、操场、雨天、考试）。
3. **开盒**：答完进入"开盲盒"动画（摇晃→脉冲→弹出），揭晓**豪意值 + 4 字母豪型代码 + 本命嘉豪标签（带稀有度发光）**。
4. **结果页**：
   - 本命嘉豪大卡（带嘉豪表情包图片）
   - 骚话评价（"全服 100 个嘉豪里，只有 3 个比你更能装"）
   - 豪意值雷达图（四个维度）
   - 分享卡（canvas 生成，可存图发朋友圈）
5. **排行榜**：豪意值云排行，按地区/年龄/豪型分榜。"你的豪意值击败了全国 87% 的嘉豪"。
6. **图鉴**：收集所有豪型，已解锁的高亮，未解锁的打码。

## 六、技术方案（确定不用框架，原生跑）

> 选原生的理由：双击即跑、中文路径不炸、能塞进一个 GitHub 仓库直接 Pages、改起来没有构建步骤。**不自作主张换 Vite/React。**

- **前端**：原生 HTML/CSS/JS 单页应用（SPA），hash 路由（`#/` `#/result` `#/ranking`），无 React/Vue/打包工具。全局挂 `window.JiaHaoTest`，模块用 IIFE 暴露，不用 ES module import（兼容 file:// 和老浏览器）。
- **云后端**：Supabase（publishable key + RLS，匿名 insert/select，不暴露 service role key）。只建一张表存做题记录 + 豪意值 + 豪型代码。**Supabase 没配好/挂了，自动回退 localStorage，单机也能跑。**
- **部署**：GitHub Pages，单独的项目仓库（**不复用 lyzbcy.github.io**），从 main 分支部署。
- **本地预览**：zeen-tools 套件（一键启动 bat + local-preview-server.js，中文路径兼容）。
- **字体**：ZCOOL KuaiLe / 站酷快乐体（童话糖果风）；数据表用系统等宽。
